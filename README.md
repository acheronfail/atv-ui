# `atv-ui`

```bash
# Install dependencies
yarn install

# Pair with AppleTV
yarn appletv pair

# Write credentials into .credentials.json
echo '"00000000-0000-0000-0000-000000000000:0000..."' > .credentials.json

# Start the server
yarn start

# Now navigate to your server's address, eg: http://192.168.1.100:8080 on your touch device
# :)
```
